Import Properties from CSV + ZIP

Upload a CSV file containing property data and a ZIP file containing the property images.

CSV File Requirements
  • ALL columns required: external_id, title, description, price, location, city, state, country, type, length, width, rooms, bedrooms, bathrooms, furnished, images
  • Images column: Comma-separated filenames (e.g., "image1.jpg,image2.jpg")
  • Property types: apartment, house, land, villa, commercial
  • Numeric fields: price, length, width, rooms, bedrooms, bathrooms must be valid numbers
  • Furnished field: true/false (case insensitive)
ZIP File Requirements
  • Max size: 50MB
  • Image formats: JPG, JPEG, PNG, WEBP
  • Max per image: 10MB
  • Max per property: 10 images
  • Structure: All images in ZIP root
